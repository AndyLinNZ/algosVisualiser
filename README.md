# algosVisualiser
Sorting visualiser

Graphical user interface for visualising sorting algorithms. Currently the algorithms included are:
1. Mergesort
2. Insertionsort
3. Quicksort
4. Bubblesort
5. Heapsort

## How to use:
1) Install python
2) Open an IDE
3) Run the program on the IDE or in the source folder enter in the terminal: `py sortingVisualiser.py`
